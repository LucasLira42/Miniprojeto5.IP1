# Meu Projeto
Um sistema simples para gerenciar finanças de uma pessoa.

## Instalação
Clone o repositório e abre por qualquer IDE que tenha uma extensão de python para aversão 3.13.9 que é a qual esse código foi escrito

## Uso
- Execute o código com python miniprojeto5\main.py
- Escolha uma das opções, é super intuitivo
